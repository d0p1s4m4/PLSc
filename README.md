## Building

### Prerequisites:

**Mandatories**: 
- a C compiler (gcc, clang, msvc....)
- GNU Make
- Git

**Optionals**:

### On UNIX-like OSes:

```
$ make
$ make check # if you want to run unit tests
```

and then as root:
```
# make install
```

### On NT-like OSes:

```
> make
```
